PDF-Tool zum massenhaften konvertieren von WORD nach PDF

Datei

Nutzen des Tools

Zuerst lädt man sich die notwendigen Dateien herunter und extrahiert diese.

Danach kopiert man die zu konvertierenden Dateien in den "IN"-Ordner.

Dann klickt man im Hauptordner (pdfconv) auf Datei >> Powershell öffnen >> Powershell hier öffnen

Als nächstes muss die Ausführung von Powershell-Scripts zugelassen werden. Mit dem Befehl 'Set-ExecutionPolicy Bypass -Scope Process' wird für den aktuellen Prozess die Ausführung jeglicher Scripte erlaubt. Dies wird noch mit der Eingabe von 'J' für Ja bestätigt.

Zuletzt wird mit '.\pdfconv.ps1' das Script aufgerufen und im Hintergrund ausgeführt.

Möglichkeiten des Tools

Das Tool nutzt die eigene Export-Funktion von Word. Dadurch bekommt man am Ende genau das als PDF im OUT-Ordner, was man beim Öffnen der Datei in Word sieht. Zusätzlich speichert das Script die Ausgabe in der log.txt - Datei, die bei der Fehlersuche helfen kann.
